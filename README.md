# Мой Плейлист - Музыкальный плеер

Веб-приложение для управления и прослушивания плейлиста с админ-панелью.

## Возможности

- Прослушивание песен с встроенным плеером
- Загрузка MP3 файлов (перетаскиванием или выбором)
- Адаптивный дизайн с градиентами
- Админ-панель для управления треками
- Удаление треков из плейлиста
- Работает на всех устройствах

## Установка и запуск

### 1. Установка бэкенда
